Итоговый проект модуля 1 (JavaRush)

Инструкция по запуску:
 - скачиваем, распаковываем, запускаем в своей IDEшке (либо клонируем по HTTPS https://github.com/MaystryukAY/CipherApp.git)
 - в файле text.txt указан изначальный текст, можно его поменять при желании
 - запускаем программу в Main, при шифровке появится файл crypted.txt, при дешифровке - decrypted.txt
